
# موقع عرض المشاريع (نسخة ثابتة بدون بناء)

هذه نسخة جاهزة لا تحتاج برمجة ولا خطوة build — فقط ارفعها على Netlify Drop.

## الملفات
- index.html (الواجهة)
- about.html (من نحن)
- projects.html (قائمة المشاريع)
- <project>.html (صفحات المشاريع)
- contact.html (التواصل)
- style.css (التصميم)

## التعديل
افتح الملفات بـ أي محرر نصوص وعدّل:
- الاسم والسطر التعريفي وبيانات التواصل داخل الصفحات.
- بيانات المشاريع والصور (تستطيع استبدال روابط الصور بروابطك).

## النشر على Netlify (الأسهل)
1) ادخل https://app.netlify.com/drop
2) اسحب **المجلد كاملاً** (portfolio_site_static) وأسقطه هناك.
3) سيظهر لك رابط مباشر مثل: https://your-site-name.netlify.app
4) تستطيع تغيير اسم الرابط من Site Settings → Change site name.

موفق!
